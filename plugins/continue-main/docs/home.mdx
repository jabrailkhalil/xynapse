## What is Xynapse?

**Xynapse enables developers to create, share, and use custom AI code agents with our open source [VS Code](https://marketplace.visualstudio.com/items?itemName=Xynapse.xynapse) and [JetBrains](https://plugins.jetbrains.com/plugin/22707-Xynapse-extension) extensions and [hub of models, rules, prompts, docs, and other building blocks](https://hub.xynapse.dev)**

## Key Features

**You can use Xynapse to build and run custom agents across your IDE, terminal, and CI.**

1. Get started with Xynapse in [VS Code](https://marketplace.visualstudio.com/items?itemName=Xynapse.xynapse) or [JetBrains](https://plugins.jetbrains.com/plugin/22707-Xynapse-extension) extensions:
- [Agent mode](/ide-extensions/agent/quick-start) to work on development tasks together with AI
- [Chat mode](/ide-extensions/chat/quick-start) to ask general questions and clarify code sections
- [Edit mode](/ide-extensions/edit/quick-start) to modify code section without leaving your current file
- [Autocomplete](/ide-extensions/autocomplete/quick-start) to receive inline code suggestions as you type

2. Try out [Xynapse CLI (cn)](https://docs.xynapse.dev/guides/cli) and give us feedback

3. Discover the models, prompts, rules, MCP tools, and agents you need to automate your workflows with AI on [Xynapse Mission Control](https://hub.xynapse.dev/)
