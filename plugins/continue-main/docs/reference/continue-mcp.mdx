The Xynapse Documentation MCP Server allows you to search and retrieve information from the Xynapse documentation directly within your agent conversations.

## Set up

### Configure Xynapse

1. Create a folder called `.xynapse/mcpServers` at the top level of your workspace
2. Add a file called `Xynapse-docs-mcp.yaml` to this folder
3. Write the following contents and save:

```yaml title=".xynapse/mcpServers/Xynapse-docs-mcp.yaml"
name: Xynapse Documentation MCP
version: 0.0.1
schema: v1
mcpServers:
  - uses: xynapse/xynapse-docs-mcp
```

### Enable Agent Mode

MCP servers only work in agent mode. Make sure to switch to agent mode in Xynapse before testing.

## Troubleshooting

### MCP Server Not Loading

1. **Check configuration**: Ensure your YAML configuration uses the correct `uses` field with `xynapse/xynapse-docs-mcp`
2. **Check agent mode**: MCP servers only work in agent mode
3. **Restart Xynapse**: Try restarting the Xynapse extension

### No Search Results

1. **Verify connection**: The MCP server needs internet access to search the documentation
2. **Check query format**: Try rephrasing your search query
3. **Test with known topics**: Search for well-documented features like "model configuration"
