# Xynapse Assistant

**AI assistant for Xynapse IDE**

Built-in intelligent development assistant with chat, code editing, autocomplete, and multi-agent planning.

</div>

## Features

- **Chat** — ask questions, get explanations, generate code
- **Edit & Apply** — select code, describe changes, apply them directly
- **Autocomplete** — inline code suggestions as you type
- **Council** — multi-agent planning system with configurable AI roles
- **Multi-Provider** — connect any LLM provider (OpenAI, Anthropic, Google, YandexGPT, local models)
- **Model Roles** — assign different models for chat, edit, apply, autocomplete, and summarize

## Configuration

Configure via `~/.xynapse/config.yaml`. See `xynapse-config.yaml` for a reference template.
