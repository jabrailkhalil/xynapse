# Xynapse キーボードショートカット

効率的な作業のための主要なショートカット：

## 基本

| アクション | Windows/Linux | macOS |
|-----------|--------------|-------|
| チャットを開く | `Ctrl+L` | `Cmd+L` |
| インライン編集 | `Ctrl+I` | `Cmd+I` |
| オートコンプリート受け入れ | `Tab` | `Tab` |
| オートコンプリート却下 | `Esc` | `Esc` |
| Diff を受け入れ | `Ctrl+Shift+Enter` | `Cmd+Shift+Enter` |
| Diff を拒否 | `Ctrl+Shift+Backspace` | `Cmd+Shift+Backspace` |
| コンテキストにコード追加 | `Ctrl+Shift+L` | `Cmd+Shift+L` |

## スラッシュコマンド

チャットで入力：
- `/edit` — 選択したコードを編集
- `/explain` — コードを説明
- `/test` — テストを生成
- `/review` — コードレビュー
- `/council` — マルチエージェント議論

## コンテキスト（@プロバイダー）

チャットで `@` を使ってコンテキストを追加：
- `@code` — 現在のファイル
- `@diff` — git の変更
- `@terminal` — ターミナル出力
- `@problems` — エラーと警告
- `@codebase` — プロジェクト検索
