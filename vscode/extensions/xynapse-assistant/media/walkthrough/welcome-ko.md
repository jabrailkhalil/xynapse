# Xynapse IDE에 오신 것을 환영합니다

Xynapse IDE는 AI 어시스턴트가 내장된 통합 개발 환경입니다.

## 기능

- **AI 채팅** — 코드에 대한 질문, 설명 받기, 솔루션 생성
- **자동완성** — 에디터에서 스마트 제안 (Tab으로 수락)
- **인라인 편집** — 코드를 선택하고 `Ctrl+I`로 AI 편집
- **프로젝트 컨텍스트** — 어시스턴트가 코드, diff, 터미널, 오류를 인식
- **Council** — 아키텍처 결정에 대한 멀티 에이전트 토론 (`/council`)
- **러시아 모델** — YandexGPT 및 GigaChat 기본 지원

## 데이터는 안전합니다

Xynapse는 IDE 서버로 데이터를 전송하지 않습니다. 모든 API 키는 `~/.xynapse/config.yaml`에 **로컬**로 저장됩니다. 텔레메트리는 완전히 비활성화되어 있습니다.

## 시작하기

1. 왼쪽의 **Xynapse** 패널 열기
2. `~/.xynapse/config.yaml`에서 모델 구성
3. `Ctrl+L`로 채팅, `Ctrl+I`로 편집, `Tab`으로 자동완성
