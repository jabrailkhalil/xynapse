# Xynapse Klavye Kisayollari

Verimli calisma icin temel kisayollari ogrenin:

## Temel

| Eylem | Windows/Linux | macOS |
|-------|--------------|-------|
| Sohbeti ac | `Ctrl+L` | `Cmd+L` |
| Satir ici duzenleme | `Ctrl+I` | `Cmd+I` |
| Otomatik tamamlamayi kabul et | `Tab` | `Tab` |
| Otomatik tamamlamayi kapat | `Esc` | `Esc` |
| Diff'i kabul et | `Ctrl+Shift+Enter` | `Cmd+Shift+Enter` |
| Diff'i reddet | `Ctrl+Shift+Backspace` | `Cmd+Shift+Backspace` |
| Baglama kod ekle | `Ctrl+Shift+L` | `Cmd+Shift+L` |

## Slash komutlari

Sohbette yazin:
- `/edit` — secili kodu duzenle
- `/explain` — kodu acikla
- `/test` — testler olustur
- `/review` — kod incelemesi
- `/council` — coklu ajan tartismasi

## Baglam (@saglayicilar)

Sohbette `@` kullanarak baglam ekleyin:
- `@code` — mevcut dosya
- `@diff` — git degisiklikleri
- `@terminal` — terminal ciktisi
- `@problems` — hatalar ve uyarilar
- `@codebase` — proje aramasi
