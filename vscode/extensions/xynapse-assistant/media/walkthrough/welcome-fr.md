# Bienvenue dans Xynapse IDE

Xynapse IDE est un environnement de developpement integre avec un assistant IA integre.

## Fonctionnalites

- **Chat IA** — posez des questions sur le code, obtenez des explications et generez des solutions
- **Autocompletion** — suggestions intelligentes dans l'editeur (Tab pour accepter)
- **Edition en ligne** — selectionnez du code et appuyez sur `Ctrl+I` pour l'edition IA
- **Contexte du projet** — l'assistant voit votre code, diff, terminal et erreurs
- **Council** — discussion multi-agents des decisions architecturales (`/council`)
- **Modeles russes** — support integre pour YandexGPT et GigaChat

## Vos donnees sont en securite

Xynapse n'envoie pas de donnees aux serveurs IDE. Toutes les cles API sont stockees **localement** dans `~/.xynapse/config.yaml`. La telemetrie est completement desactivee.

## Pour commencer

1. Ouvrez le panneau lateral **Xynapse** (icone a gauche)
2. Configurez les modeles dans `~/.xynapse/config.yaml`
3. Utilisez `Ctrl+L` pour le chat, `Ctrl+I` pour l'edition, `Tab` pour l'autocompletion
