# Xynapse IDE へようこそ

Xynapse IDE は AI アシスタントを内蔵した統合開発環境です。

## 機能

- **AI チャット** — コードに関する質問、説明の取得、ソリューションの生成
- **オートコンプリート** — エディタ内のスマートな提案（Tab で受け入れ）
- **インライン編集** — コードを選択して `Ctrl+I` で AI 編集
- **プロジェクトコンテキスト** — アシスタントはコード、diff、ターミナル、エラーを認識
- **Council** — アーキテクチャ決定のマルチエージェント議論（`/council`）
- **ロシアのモデル** — YandexGPT と GigaChat のビルトインサポート

## データは安全です

Xynapse は IDE サーバーにデータを送信しません。すべての API キーは `~/.xynapse/config.yaml` に**ローカル**で保存されます。テレメトリは完全に無効化されています。

## 始め方

1. 左側の **Xynapse** パネルを開く
2. `~/.xynapse/config.yaml` でモデルを設定
3. `Ctrl+L` でチャット、`Ctrl+I` で編集、`Tab` でオートコンプリート
