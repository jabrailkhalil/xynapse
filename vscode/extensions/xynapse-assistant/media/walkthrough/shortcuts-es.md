# Atajos de teclado de Xynapse

Aprenda los atajos clave para trabajar eficientemente:

## Esenciales

| Accion | Windows/Linux | macOS |
|--------|--------------|-------|
| Abrir chat | `Ctrl+L` | `Cmd+L` |
| Edicion en linea | `Ctrl+I` | `Cmd+I` |
| Aceptar autocompletado | `Tab` | `Tab` |
| Descartar autocompletado | `Esc` | `Esc` |
| Aceptar diff | `Ctrl+Shift+Enter` | `Cmd+Shift+Enter` |
| Rechazar diff | `Ctrl+Shift+Backspace` | `Cmd+Shift+Backspace` |
| Agregar codigo al contexto | `Ctrl+Shift+L` | `Cmd+Shift+L` |

## Comandos slash

Escriba en el chat:
- `/edit` ‚Äî editar codigo seleccionado
- `/explain` ‚Äî explicar codigo
- `/test` ‚Äî generar pruebas
- `/review` ‚Äî revision de codigo
- `/council` ‚Äî discusion multiagente

## Contexto (@proveedores)

Use `@` en el chat para agregar contexto:
- `@code` ‚Äî archivo actual
- `@diff` ‚Äî cambios git
- `@terminal` ‚Äî salida del terminal
- `@problems` ‚Äî errores y advertencias
- `@codebase` ‚Äî busqueda en el proyecto
