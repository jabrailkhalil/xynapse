# Bienvenido a Xynapse IDE

Xynapse IDE es un entorno de desarrollo integrado con un asistente de IA incorporado.

## Funcionalidades

- **Chat con IA** — haga preguntas sobre el codigo, obtenga explicaciones y genere soluciones
- **Autocompletado** — sugerencias inteligentes en el editor (Tab para aceptar)
- **Edicion en linea** — seleccione codigo y presione `Ctrl+I` para edicion con IA
- **Contexto del proyecto** — el asistente ve su codigo, diff, terminal y errores
- **Council** — discusion multiagente de decisiones arquitectonicas (`/council`)
- **Modelos rusos** — soporte integrado para YandexGPT y GigaChat

## Sus datos estan seguros

Xynapse no envia datos a servidores IDE. Todas las claves API se almacenan **localmente** en `~/.xynapse/config.yaml`. La telemetria esta completamente desactivada.

## Primeros pasos

1. Abra el panel lateral **Xynapse** (icono a la izquierda)
2. Configure modelos en `~/.xynapse/config.yaml`
3. Use `Ctrl+L` para chat, `Ctrl+I` para edicion, `Tab` para autocompletado
