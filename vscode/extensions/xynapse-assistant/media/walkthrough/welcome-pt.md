# Bem-vindo ao Xynapse IDE

Xynapse IDE e um ambiente de desenvolvimento integrado com assistente de IA incorporado.

## Funcionalidades

- **Chat com IA** — faca perguntas sobre codigo, obtenha explicacoes e gere solucoes
- **Autocompletar** — sugestoes inteligentes no editor (Tab para aceitar)
- **Edicao inline** — selecione codigo e pressione `Ctrl+I` para edicao com IA
- **Contexto do projeto** — o assistente ve seu codigo, diff, terminal e erros
- **Council** — discussao multiagente de decisoes arquiteturais (`/council`)
- **Modelos russos** — suporte integrado para YandexGPT e GigaChat

## Seus dados estao seguros

Xynapse nao envia dados para servidores IDE. Todas as chaves API sao armazenadas **localmente** em `~/.xynapse/config.yaml`. A telemetria esta completamente desativada.

## Primeiros passos

1. Abra o painel lateral **Xynapse** (icone a esquerda)
2. Configure modelos em `~/.xynapse/config.yaml`
3. Use `Ctrl+L` para chat, `Ctrl+I` para edicao, `Tab` para autocompletar
