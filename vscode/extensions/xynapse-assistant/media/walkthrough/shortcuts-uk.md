# Гарячi клавiшi Xynapse

Запам'ятайте основнi сполучення для ефективної роботи:

## Основнi

| Дiя | Windows/Linux | macOS |
|-----|--------------|-------|
| Вiдкрити чат | `Ctrl+L` | `Cmd+L` |
| Inline-редагування | `Ctrl+I` | `Cmd+I` |
| Прийняти автодоповнення | `Tab` | `Tab` |
| Вiдхилити автодоповнення | `Esc` | `Esc` |
| Прийняти diff | `Ctrl+Shift+Enter` | `Cmd+Shift+Enter` |
| Вiдхилити diff | `Ctrl+Shift+Backspace` | `Cmd+Shift+Backspace` |
| Додати код до контексту | `Ctrl+Shift+L` | `Cmd+Shift+L` |

## Slash-команди

Введiть у чат:
- `/edit` — редагувати видiлений код
- `/explain` — пояснити код
- `/test` — згенерувати тести
- `/review` — огляд коду
- `/council` — мультиагентне обговорення

## Контекст (@провайдери)

Використовуйте `@` у чатi для додавання контексту:
- `@code` — поточний файл
- `@diff` — git-змiни
- `@terminal` — вивiд термiналу
- `@problems` — помилки та попередження
- `@codebase` — пошук по проекту
