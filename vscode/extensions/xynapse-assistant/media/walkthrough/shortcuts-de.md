# Xynapse Tastenkurzel

Lernen Sie die wichtigsten Kurzel fur effizientes Arbeiten:

## Grundlegende

| Aktion | Windows/Linux | macOS |
|--------|--------------|-------|
| Chat offnen | `Ctrl+L` | `Cmd+L` |
| Inline-Bearbeitung | `Ctrl+I` | `Cmd+I` |
| Autovervollstandigung akzeptieren | `Tab` | `Tab` |
| Autovervollstandigung ablehnen | `Esc` | `Esc` |
| Diff akzeptieren | `Ctrl+Shift+Enter` | `Cmd+Shift+Enter` |
| Diff ablehnen | `Ctrl+Shift+Backspace` | `Cmd+Shift+Backspace` |
| Code zum Kontext hinzufugen | `Ctrl+Shift+L` | `Cmd+Shift+L` |

## Slash-Befehle

Geben Sie im Chat ein:
- `/edit` — ausgewahlten Code bearbeiten
- `/explain` — Code erklaren
- `/test` — Tests generieren
- `/review` — Code-Review
- `/council` — Multi-Agenten-Diskussion

## Kontext (@-Anbieter)

Verwenden Sie `@` im Chat fur Kontext:
- `@code` — aktuelle Datei
- `@diff` — Git-Anderungen
- `@terminal` — Terminal-Ausgabe
- `@problems` — Fehler und Warnungen
- `@codebase` — Projektsuche
