# Xynapse 키보드 단축키

효율적인 작업을 위한 주요 단축키:

## 필수

| 동작 | Windows/Linux | macOS |
|------|--------------|-------|
| 채팅 열기 | `Ctrl+L` | `Cmd+L` |
| 인라인 편집 | `Ctrl+I` | `Cmd+I` |
| 자동완성 수락 | `Tab` | `Tab` |
| 자동완성 무시 | `Esc` | `Esc` |
| Diff 수락 | `Ctrl+Shift+Enter` | `Cmd+Shift+Enter` |
| Diff 거부 | `Ctrl+Shift+Backspace` | `Cmd+Shift+Backspace` |
| 컨텍스트에 코드 추가 | `Ctrl+Shift+L` | `Cmd+Shift+L` |

## 슬래시 명령

채팅에서 입력:
- `/edit` — 선택한 코드 편집
- `/explain` — 코드 설명
- `/test` — 테스트 생성
- `/review` — 코드 리뷰
- `/council` — 멀티 에이전트 토론

## 컨텍스트 (@공급자)

채팅에서 `@`를 사용하여 컨텍스트 추가:
- `@code` — 현재 파일
- `@diff` — git 변경사항
- `@terminal` — 터미널 출력
- `@problems` — 오류 및 경고
- `@codebase` — 프로젝트 검색
