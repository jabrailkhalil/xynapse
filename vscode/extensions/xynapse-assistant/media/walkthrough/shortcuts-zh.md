# Xynapse 键盘快捷键

学习高效工作的关键快捷键：

## 基本操作

| 操作 | Windows/Linux | macOS |
|------|--------------|-------|
| 打开聊天 | `Ctrl+L` | `Cmd+L` |
| 内联编辑 | `Ctrl+I` | `Cmd+I` |
| 接受自动补全 | `Tab` | `Tab` |
| 取消自动补全 | `Esc` | `Esc` |
| 接受差异 | `Ctrl+Shift+Enter` | `Cmd+Shift+Enter` |
| 拒绝差异 | `Ctrl+Shift+Backspace` | `Cmd+Shift+Backspace` |
| 添加代码到上下文 | `Ctrl+Shift+L` | `Cmd+Shift+L` |

## 斜杠命令

在聊天中输入：
- `/edit` — 编辑选中的代码
- `/explain` — 解释代码
- `/test` — 生成测试
- `/review` — 代码审查
- `/council` — 多智能体讨论

## 上下文（@提供者）

在聊天中使用 `@` 添加上下文：
- `@code` — 当前文件
- `@diff` — git 更改
- `@terminal` — 终端输出
- `@problems` — 错误和警告
- `@codebase` — 项目搜索
