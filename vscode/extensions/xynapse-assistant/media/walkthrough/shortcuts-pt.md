# Atalhos de teclado do Xynapse

Aprenda os atalhos principais para trabalhar eficientemente:

## Essenciais

| Acao | Windows/Linux | macOS |
|------|--------------|-------|
| Abrir chat | `Ctrl+L` | `Cmd+L` |
| Edicao inline | `Ctrl+I` | `Cmd+I` |
| Aceitar autocompletar | `Tab` | `Tab` |
| Descartar autocompletar | `Esc` | `Esc` |
| Aceitar diff | `Ctrl+Shift+Enter` | `Cmd+Shift+Enter` |
| Rejeitar diff | `Ctrl+Shift+Backspace` | `Cmd+Shift+Backspace` |
| Adicionar codigo ao contexto | `Ctrl+Shift+L` | `Cmd+Shift+L` |

## Comandos slash

Digite no chat:
- `/edit` тАФ editar codigo selecionado
- `/explain` тАФ explicar codigo
- `/test` тАФ gerar testes
- `/review` тАФ revisao de codigo
- `/council` тАФ discussao multiagente

## Contexto (@provedores)

Use `@` no chat para adicionar contexto:
- `@code` тАФ arquivo atual
- `@diff` тАФ alteracoes git
- `@terminal` тАФ saida do terminal
- `@problems` тАФ erros e avisos
- `@codebase` тАФ busca no projeto
